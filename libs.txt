Insidelog uses one library that is not part of Java standard platform.

- "juniversalchardet" is a Java port of 'universalchardet', that is the encoding detector library of Mozilla

	Source code can be found at following link: 
	https://code.google.com/p/juniversalchardet/

- "JSch" is a Java port of SSH protocol made by JCraft
	Source code can be found at following link: 
	http://www.jcraft.com/jsch/
